HKEY Property Editor
ReadMe

Description

PJHKEYPropEdit.pas defines a property editor named TPJHKEYPropEditor for properties of type HKEY – i.e. registry root keys. The editor causes a drop-down list of all the registry root keys supported by Delphi to be displayed in the object inspector for any property of type HKEY.

Such a property is exposed by the TPJRegWdwState component.

Compatibility

The property editor compiles and works with all versions of Delphi from Delphi 2. It has been tested with Delphi 2, 3, 4, 5, 7, 2006 and 2010.

Installation

The HKEY Property Editor is supplied in a zip file. Before installing you need to extract all the files from the zip file. The following files will be extracted:

  • PJHKEYPropEdit.pas – Property editor source code.
  • ReadMe.htm – This read-me file.
  • ChangeLog.txt – Change log.
  • MPL.txt – Mozilla Public License.

You can now intall the property editor into the Delphi IDE.

For Delphi 3 onwards you must include PJHKEYPropEdit.pas in a design time package that is installed into the Delphi IDE. If you need help doing this see here.

For Delphi 2 install PJHKEYPropEdit.pas as if it were a component.

Update History

A complete change log is provided in the file ChangeLog.txt that is included in the download.
